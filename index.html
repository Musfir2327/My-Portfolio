Projects

Smart Clinic Management System

ProClinic is a modern clinic management system designed to streamline healthcare administration with efficiency and ease. Built using Spring Boot for the backend, React for the frontend, and MongoDB for data handling, it offers a seamless experience for managing appointments, billing, and patient records. Key features include appointment scheduling with automated reminders, a comprehensive billing system, and voice-to-text functionality for quick note-taking. With a user-friendly interface and a secure, scalable architecture, ProClinic enhances workflow for doctors and staff, ensuring better patient engagement and efficient clinic operations.

Library Book Stock Management System (LBMS)

Library Book Management System (LBMS) is a robust application designed to streamline library operations by efficiently managing book records. Developed using Spring Boot for the backend, Angular for the frontend, and MySQL for secure data storage, LBMS allows users to add, update, and delete books seamlessly. With a user-friendly interface, efficient search and filtering, and scalable architecture, it ensures smooth and organized book management, enhancing accessibility and usability for librarians and users alike

Hotel Management System

The Hotel Management System is a comprehensive solution designed to streamline hotel operations with efficiency and ease. Built using Spring Boot for the backend, React.js for the frontend, and MySQL for secure data management, it enables seamless inventory management, order processing, and user authentication. With real-time data synchronization and a responsive user interface, the system ensures smooth service delivery, accurate tracking of resources, and enhanced user experience for hotel staff and administrators.
